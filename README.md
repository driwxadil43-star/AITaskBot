# AITaskBot

**وصف المشروع:**

`AITaskBot` هو مساعد مهام بسيط يعتمد على الذكاء الاصطناعي (AI) مصمم لمساعدتك في إدارة مهامك اليومية بكفاءة. يتيح لك هذا البوت إضافة المهام، عرض قائمة المهام الحالية، والخروج من البرنامج. تم تطويره باستخدام بايثون ويهدف إلى أن يكون أساساً لمشروع إدارة مهام أكثر تعقيداً في المستقبل.

**الميزات الحالية:**

*   **إضافة المهام:** يمكنك إضافة مهام جديدة إلى قائمتك.
*   **عرض المهام:** يمكنك عرض جميع المهام المضافة حالياً.
*   **واجهة سطر الأوامر:** تفاعل بسيط ومباشر عبر سطر الأوامر.

**كيفية التشغيل:**

1.  **استنساخ المستودع:**
    ```bash
    git clone https://github.com/driwxadil43-star/AITaskBot.git
    cd AITaskBot
    ```
2.  **تشغيل البوت:**
    ```bash
    python3 bot.py
    ```

**الخطط المستقبلية (Issues):**

نحن نعمل باستمرار على تحسين `AITaskBot` وإضافة ميزات جديدة. إليك بعض الخطط المستقبلية التي سيتم العمل عليها:

*   **إضافة ميزة حفظ المهام في ملف:** لضمان عدم فقدان المهام عند إغلاق البرنامج، سيتم إضافة وظيفة لحفظ المهام في ملف (نصي أو JSON) وتحميلها عند بدء التشغيل. [Issue #1](https://github.com/driwxadil43-star/AITaskBot/issues/1)
*   **إضافة تواريخ استحقاق للمهام:** سيتم تمكين المستخدمين من تحديد تواريخ استحقاق لكل مهمة وتلقي تنبيهات عند اقتراب الموعد. [Issue #2](https://github.com/driwxadil43-star/AITaskBot/issues/2)
*   **تحسين واجهة المستخدم:** سيتم تحسين تجربة المستخدم من خلال جعل الواجهة أكثر تفاعلية وجاذبية باستخدام مكتبات بايثون المخصصة لواجهة سطر الأوامر. [Issue #3](https://github.com/driwxadil43-star/AITaskBot/issues/3)
